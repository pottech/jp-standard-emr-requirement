# 概要・背景

## 概要

オンライン資格確認等システムの導入に当たり、オンライン資格確認等システムが提供する機能及び医療機関・薬局のシステムベンダが提供しているシステムに実装いただきたい内容等について記載した技術解説書です。

## サブシステム構成

オンライン資格確認等システムは以下の4つのサブシステムから構成されます。

| サブシステム | 説明 |
|---|---|
| オンライン資格確認システム | 加入者（患者）が医療機関・薬局を受診する際、マイナンバーカードの利用者証明用電子証明書、または資格確認書を用いて、オンラインで資格確認を行う |
| 薬剤情報閲覧システム | レセプトをもとに薬剤情報を管理する機能。加入者の同意がある場合、医療機関・薬局にて薬剤情報を閲覧できる |
| 特定健診情報閲覧システム | 医療保険者等からの特定健診情報の報告をもとに管理する機能。加入者の同意がある場合、医療機関・薬局にて閲覧できる（後期広域連合の健康診査も対象） |
| レセプト振替システム | レセプト情報に記載された保険者番号等を利用して、資格情報の有効性を確認し、受診日・調剤日時点の資格情報に基づいてレセプト請求先を振替・分割する |

## 対象施設・対象者

- **対象施設**: 全国の保険医療機関・保険薬局
- **対象保険者**: 全国健康保険協会、健康保険組合、国民健康保険組合、後期高齢者医療広域連合、国家公務員共済組合、地方公務員共済組合、日本私立学校振興・共済事業団、市町村国民健康保険

## 運用開始時期

| サービス | 運用開始時期 |
|---|---|
| 資格確認 | 令和3年3月目途 |
| 特定健診情報閲覧 | 令和3年3月目途 |
| 薬剤情報閲覧 | 令和3年10月目途 |
| レセプト振替 | 令和3年10月目途 |

## レスポンスタイム

資格確認の要求を取得してから照会結果を取得するまで3〜7秒の想定。薬剤情報/特定健診情報についても同様に3〜7秒の想定。

## ソフトウェア構成

| ソフトウェア | 提供元 | 主な機能 |
|---|---|---|
| オンライン資格確認等連携ソフト | 支払基金 | 被保険者番号を利用した資格確認、照会番号の登録、薬剤情報・特定健診情報の取得等 |
| マイナンバーカード処理ソフト | 支払基金 | PIN無し認証等を行うマイナンバーカードを利用した資格確認や健康保険証利用の申込（初回登録） |
| 本人認証用カードリーダーソフト | カードリーダー製造ベンダ | カードリーダーを利用した本人認証（顔認証や暗証番号入力による本人認証） |

## 薬剤情報・特定健診情報の閲覧

薬剤情報・特定健診情報の閲覧に際しては、以下のアクセス制御が必要です。

- 同意意思を明示的に確認した上でマイナンバーカードをタッチする仕組みの実装
- 有資格者等（医師、歯科医師、薬剤師等）のみ取得できる仕組みの実装（ID・PW認証）
- 資格確認端末におけるデータ管理として一定時間で削除する仕組みの実装
- いずれの医療機関・薬局が取得したかが後日調査可能となるログ管理の仕組みの実装
