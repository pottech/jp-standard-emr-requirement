# 認証方式

オンライン資格確認等システムでは、マイナンバーカード（物理カード・スマートフォン）および資格確認書による認証方式をサポートしています。

## マイナンバーカード（物理カード）

### PIN無し認証

暗証番号入力を求めない認証方式です。マイナンバーカードをかざすだけで利用者証明用電子証明書の読み取りを行います。

<WorkflowStep order={1} actor="顔認証付きカードリーダー / 窓口" action="本人確認">
  顔認証付きカードリーダーで患者の顔とマイナンバーカードのICチップ内の顔画像を照合し認証、または窓口での顔写真の目視確認を行う。
</WorkflowStep>

<WorkflowStep order={2} actor="資格確認端末" action="特定機関認証">
  特定機関認証を行い、検証機関認証用電子証明書の送付とPIN無し認証処理を実施する。
</WorkflowStep>

<WorkflowStep order={3} actor="オンライン資格確認等システム" action="電子証明書の有効性確認">
  利用者証明用電子証明書の有効性確認処理を実施する。
</WorkflowStep>

<WorkflowStep order={4} actor="オンライン資格確認等システム" action="資格情報の返却">
  資格情報等のステータス（有効/無効）と資格情報を返却する。資格確認端末で受信・格納し、レセプトコンピュータ等と連携する。
</WorkflowStep>

#### カードリーダーの種類

| カードリーダー | 説明 |
|---|---|
| 顔認証付きカードリーダー | マイナンバーカードの写真データを用いて顔認証を行うカードリーダー。患者側に向けて設置する必要がある |
| 汎用カードリーダー | 公的個人認証サービスに対応したICカードリーダー。目視で顔確認を行う場合に使用。PIN認証を行う場合、テンキー等の附属が必要 |

#### 必要な準備

- 顔認証付きカードリーダーまたは汎用カードリーダー
- マイナンバーカード処理ソフト、本人認証用カードリーダーソフト、オンライン資格確認等連携ソフトのインストール
- 健康保険証利用の申込（初回登録）

### PIN認証

暗証番号入力を求める認証方式です。

<WorkflowStep order={1} actor="患者" action="暗証番号入力">
  患者がカードリーダーに暗証番号（4桁）を入力する。
</WorkflowStep>

<WorkflowStep order={2} actor="マイナンバーカード処理ソフト" action="資格確認API呼び出し">
  「暗証番号入力による資格確認API」を呼び出し、利用者証明用電子証明書の有効性を確認する。
</WorkflowStep>

<WorkflowStep order={3} actor="オンライン資格確認等システム" action="資格情報取得">
  利用者証明用電子証明書が有効な場合は、シリアル番号と紐づく資格情報を取得し返却する。
</WorkflowStep>

<WorkflowStep order={4} actor="資格確認端末" action="ファイル出力">
  取得した資格情報はレセプトコンピュータ等との連携フォルダにファイルで出力する。
</WorkflowStep>

## スマートフォンのマイナンバーカード

「Androidスマホ用電子証明書搭載サービス」及び「iPhoneのマイナンバーカード」のサービスによって、スマートフォンに追加して利用できるマイナンバーカードです。

### Android

暗証番号（4桁）でAndroidスマホ用電子証明書から利用者証明用電子証明書を取り出し、有効性を確認します。

<WorkflowStep order={1} actor="患者" action="暗証番号入力">
  患者がAndroidスマホ用電子証明書の利用者証明用暗証番号を入力する。
</WorkflowStep>

<WorkflowStep order={2} actor="本人認証用カードリーダーソフト" action="PIN認証要求">
  本人認証用カードリーダーソフトがPIN認証を要求し、マイナンバーカード処理ソフトによるPIN認証処理を実施する。
</WorkflowStep>

<WorkflowStep order={3} actor="オンライン資格確認等システム" action="電子証明書の有効性確認・資格情報返却">
  利用者証明用電子証明書の有効性確認処理を行い、資格情報等のステータスと資格情報を返却する。
</WorkflowStep>

> **注記**: スマートフォンのマイナンバーカードの読み取りに対応した顔認証付きカードリーダーが必要です。対応していない場合は汎用カードリーダーとのペアリング設定が必要です。

### iPhone

iPhoneの生体認証（Face ID/Touch ID）により利用者証明用電子証明書を取り出し、オンライン資格確認等システムに登録されている資格情報を照会します。

<WorkflowStep order={1} actor="患者" action="生体認証">
  患者がiPhoneの生体認証（Face ID/Touch ID）で利用者証明用電子証明書を取り出す。
</WorkflowStep>

<WorkflowStep order={2} actor="マイナンバーカード処理ソフト" action="認証処理">
  本人認証用カードリーダーソフトおよびマイナンバーカード処理ソフトが認証処理を実施する。
</WorkflowStep>

<WorkflowStep order={3} actor="オンライン資格確認等システム" action="電子証明書の有効性確認・資格情報返却">
  利用者証明用電子証明書の有効性確認処理を行い、資格情報等のステータスと資格情報を返却する。
</WorkflowStep>

## 資格確認書

マイナンバーカードを保持しない被保険者・被扶養者が使用する認証方式です。

### 新患時

<WorkflowStep order={1} actor="レセプトコンピュータ" action="照会情報入力">
  レセプトコンピュータにて照会情報（保険者番号、被保険者資格に係る記号・番号、枝番、生年月日）を入力する。
</WorkflowStep>

<WorkflowStep order={2} actor="オンライン資格確認等システム" action="資格情報照会">
  照会した資格情報等を資格確認端末に一時格納する。
</WorkflowStep>

<WorkflowStep order={3} actor="レセプトコンピュータ" action="資格情報取得">
  レセプトコンピュータから資格確認端末へ資格情報等のリクエストをして取得する。
</WorkflowStep>

> **注記**: 資格確認書に枝番の記載が無い場合は、入力しなくても照会可能です。

### 2回目以降

レセプトコンピュータにて当該患者カルテを表示し、資格情報等の照会を行います。照会した資格情報等を資格確認端末に一時格納し、レセプトコンピュータから取得します。

### 一括照会

予約した患者の資格情報の有効性を事前に照会したい場合に、複数の患者の被保険者番号で一括して照会できます。

> **注記**: 一括照会を実施した場合、照会した被保険者番号に対応する資格情報が有効かどうかを確認できるものであり、最新の資格情報まで提供するものではありません。

## 照会結果パターン

| 条件 | マイナンバーカード | 資格確認書 | 一括照会 |
|---|---|---|---|
| 資格が有効な場合 | その時点で登録されている資格情報を表示 | 照会した記号・番号に対応する資格情報を表示 | 照会した記号・番号に対応する資格情報を表示 |
| 資格が喪失し新しい資格が未登録 | 現時点で資格喪失である旨を表示（喪失している直近の資格情報を参考情報として表示） | 対応する資格が喪失している旨を表示 | 対応する資格が喪失している旨を表示 |
| 資格が喪失し新しい資格が登録済み | （該当なし） | 対応する資格が喪失しており、新しい資格情報がある旨を表示 | 対応する資格が喪失しており、新しい資格情報がある旨を表示 |

## 限度額適用認定証等

患者の本人確認後、本人の同意を得たうえで照会が可能です。

| 証の種類 | 表示内容 | 取得方法 |
|---|---|---|
| 限度額適用認定証 | 適用区分 | マイナンバーカード: 同意不要、資格確認書: 同意必要 |
| 限度額適用・標準負担額減額認定証 | 適用区分（長期入院該当年月日） | 同上 |
| 標準負担額減額認定証 | 適用区分（長期入院該当年月日） | 同上 |
| 特定疾病療養受療証 | 認定疾病名（自己負担限度額） | マイナンバーカードで本人確認し、本人が同意した場合のみ |

## 健康保険証利用の申込

マイナンバーカードの健康保険証利用に当たっては、加入者がマイナポータルにアクセスして、健康保険証利用の申込（初回登録）を行う必要があります。健康保険証利用の申込をしていない患者が実物のマイナンバーカードを利用して受診した場合、顔認証付きカードリーダーによる顔認証または暗証番号入力を行うことにより、医療機関・薬局でも健康保険証利用の申込（初回登録）ができます。
