# 電子カルテ・レセコン標準仕様ガイド

国が定めた電子カルテ及びレセプトコンピュータの標準要件を、分かりやすく構造化して公開するサイトです。

## 背景

医療DX推進の一環として、電子カルテおよびレセプトコンピュータに関する標準仕様書が策定されました。本サイトでは、これらの仕様書の要件を体系的に整理し、システムベンダや医療機関が参照しやすい形で提供しています。

## コンテンツ一覧

### 電子カルテ標準仕様

医科診療所向け・中小病院向けの電子カルテ標準仕様書に基づく要件を整理しています。

- **機能要件**: 医療DXサービス群との接続機能、電子処方箋関連機能等
- **非機能要件**: 可用性、セキュリティ、データ保管等
- **アーキテクチャ**: クラウドネイティブ、SaaS型、マルチテナント方式等
- **データ移行**: 電子カルテ間のデータ移行要件
- **システム連携**: 外部インターフェイス仕様、API仕様等
- **情報提供・公開**: 価格公開、セキュリティ開示等

### レセプトコンピュータ標準仕様

医科診療所向け・中小病院向けのレセプトコンピュータ標準仕様書に基づく要件を整理しています。

### 電子処方箋管理サービス

電子処方箋管理サービスの導入に関するシステムベンダ向け技術解説書に基づき、API仕様、ワークフロー、電子署名の仕組みを解説しています。

### オンライン資格確認等システム

オンライン資格確認等システムの導入に関するシステムベンダ向け技術解説書に基づき、認証方式、ネットワーク構成、セキュリティ対策を解説しています。

### 主治医意見書電送サービス

主治医意見書／請求書電送サービスの導入に関するシステムベンダ向け技術解説書に基づき、PMH・介護情報基盤を活用した主治医意見書・請求書の電子送受信サービスのシステム機能、API仕様、ネットワーク構成、セキュリティ対策を解説しています。

### ウェブアクセシビリティ

デジタル庁が発行したウェブアクセシビリティ導入ガイドブックに基づき、WCAG/JIS規格の達成基準、実践プロセス、チェック項目を解説しています。医療情報システムの開発においても重要なアクセシビリティ対応の指針を提供します。

### 横断的コンテンツ

- **用語集**: 全文書の用語を統合し、50音順に整理した用語集
- **比較表**: 医科診療所向けと中小病院向けの要件差分を一覧化

## 対象文書

本サイトは以下の文書に基づいています:

1. 医科診療所向け電子カルテ及びレセプトコンピュータ標準仕様書（基本要件）【第1.0版】
2. 中小病院向け電子カルテ及びレセプトコンピュータ標準仕様書（基本要件）【第1.0版】
3. 電子処方箋管理サービスの導入に関するシステムベンダ向け技術解説書【医療機関・薬局】
4. オンライン資格確認等システムの導入に関するシステムベンダ向け技術解説書【医療機関・薬局】
5. 主治医意見書／請求書 電送サービスの導入に関するシステムベンダ向け技術解説書
