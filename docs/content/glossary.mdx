# 用語集

電子カルテ・レセプトコンピュータ標準仕様、電子処方箋管理サービス、オンライン資格確認等システム、主治医意見書電送サービス、ウェブアクセシビリティに関する用語集です。

<GlossarySearch />

<GlossaryEntry term="ISMS" reading="あいえすえむえす" englishTerm="ISMS (Information Security Management System)" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  情報セキュリティマネジメントシステム（Information Security Management System）。ISO/IEC 27001に基づく認証制度。電子カルテの非機能要件において、ISMS認証及びISMSクラウドセキュリティ認証の取得がセキュリティ要件として規定される。
</GlossaryEntry>

<GlossaryEntry term="ICカード" reading="あいしーかーど" englishTerm="IC Card" source="電子処方箋技術解説書">
  医師・薬剤師が電子署名を行う方法として、ローカル署名やカードレス署名を利用する際に使用する物理的なカード。ローカル署名、カードレス署名で利用できるICカードとしてHPKIカードがあり、カードレス署名ではマイナンバーカードを活用することも可能。
</GlossaryEntry>

<GlossaryEntry term="IPsec+IKE接続方式" reading="あいぴーせっくぷらすあいけーいーせつぞくほうしき" englishTerm="IPsec+IKE Connection Method" source="電子処方箋技術解説書 / オンライン資格確認等技術解説書">
  インターネットを経由してIPsecトンネルにより安全な接続を確保する方式。ルーター型、クライアント型/PCキー型/USBキー型がある。IPv4での接続方式。
</GlossaryEntry>

<GlossaryEntry term="IP-VPN接続方式" reading="あいぴーぶいぴーえぬせつぞくほうしき" englishTerm="IP-VPN Connection Method" source="電子処方箋技術解説書 / オンライン資格確認等技術解説書">
  閉域IP網を利用した接続方式。レセプトのオンライン請求で実績のあるオンライン請求ネットワークを利用する。IPv4（オンライン請求）とIPv6（オンライン資格確認等）を併用。光回線に限る。
</GlossaryEntry>

<GlossaryEntry term="アクセシビリティ" reading="あくせしびりてぃ" englishTerm="Accessibility" source="ウェブアクセシビリティ導入ガイドブック">
  「近づくことができる」という意味で、「ウェブや情報そのものへ到達できること」や「製品やサービスを利用できること」という意味で使われる。一般的に「障害者・高齢者を含む利用者のアクセスしやすさ」と説明される。
</GlossaryEntry>

<GlossaryEntry term="アクセシビリティ・オーバーレイ" reading="あくせしびりてぃおーばーれい" englishTerm="Accessibility Overlay" source="ウェブアクセシビリティ導入ガイドブック">
  ウェブサイトやウェブシステムにスクリプトを追加することで、アクセシビリティを向上しようとする仕組み。JavaScriptで実装されることがほとんど。使用は非推奨。
</GlossaryEntry>

<GlossaryEntry term="アクセストークン" reading="あくせすとーくん" englishTerm="Access Token" source="電子処方箋技術解説書">
  システムに接続する端末が認証済みであることを識別するための情報。電子処方箋においては、カードレス署名を行う場合、鍵管理サービスにアクセストークンを発行してもらった上で、鍵管理サービスで管理される電子署名を用いた電子署名を行う。
</GlossaryEntry>

<GlossaryEntry term="医師" reading="いし" englishTerm="Physician / Doctor" source="電子処方箋技術解説書">
  医師、歯科医師を指す。
</GlossaryEntry>

<GlossaryEntry term="ISMAP" reading="いすまっぷ" englishTerm="ISMAP (Information system Security Management and Assessment Program)" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  政府情報システムのためのセキュリティ評価制度（Information system Security Management and Assessment Program）。電子カルテの非機能要件において、ISMAP認証またはISMS認証及びISMSクラウドセキュリティ認証の取得が求められる。
</GlossaryEntry>

<GlossaryEntry term="一部準拠" reading="いちぶじゅんきょ" englishTerm="Partial Compliance" source="ウェブアクセシビリティ導入ガイドブック">
  JIS X 8341-3:2016の対応度表記で、達成基準の一部を満たしている場合に使える表記。一部準拠の場合は追加で今後の対応方針を記載する。
</GlossaryEntry>

<GlossaryEntry term="医薬品マスタ" reading="いやくひんますた" englishTerm="Drug Master" source="電子処方箋技術解説書">
  電子処方箋管理サービスと、電子処方箋管理サービスを利用する医療機関・薬局システムで利用する医薬品についてのマスタ。
</GlossaryEntry>

<GlossaryEntry term="医療機関・薬局のシステム" reading="いりょうきかん・やっきょくのしすてむ" englishTerm="Medical Institution and Pharmacy Systems" source="オンライン資格確認等技術解説書">
  レセプトコンピュータ／医事会計システム、電子カルテシステム、調剤システムの総称。
</GlossaryEntry>

<GlossaryEntry term="医療情報セキュリティ開示書（SDS）" reading="いりょうじょうほうせきゅりてぃかいじしょ（えすでぃーえす）" englishTerm="Medical Information Security Disclosure Statement (SDS)" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  電子カルテベンダが医療機関に対して開示するセキュリティ関連情報をまとめた文書。標準仕様書では、医療機関等の要請があった場合にSDS Ver5.0を開示することが遵守要件として規定されている。
</GlossaryEntry>

<GlossaryEntry term="医療DX" reading="いりょうでぃーえっくす" englishTerm="Medical DX (Digital Transformation)" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  医療分野におけるデジタルトランスフォーメーション。国が推進する施策として、電子カルテの標準化、オンライン資格確認、電子処方箋、電子カルテ情報共有サービス、共通算定モジュール、介護情報基盤との連携等が含まれる。
</GlossaryEntry>

<GlossaryEntry term="医療保険者等" reading="いりょうほけんしゃとう" englishTerm="Medical Insurers" source="オンライン資格確認等技術解説書">
  全国健康保険協会、健康保険組合、国民健康保険組合、後期高齢者医療広域連合、市町村国民健康保険、国家公務員共済組合、地方公務員共済組合、及び日本私立学校振興・共済事業団の総称。
</GlossaryEntry>

<GlossaryEntry term="院内処方等ID" reading="いんないしょほうとうあいでぃー" englishTerm="In-Hospital Prescription ID" source="電子処方箋技術解説書">
  院内処方等情報提供ファイルを電子処方箋管理サービスに登録し、発行されるID。電子処方箋管理サービス内の院内処方等情報提供ファイルを管理するため、一意に採番される。
</GlossaryEntry>

<GlossaryEntry term="院内処方等情報" reading="いんないしょほうとうじょうほう" englishTerm="In-Hospital Prescription Information" source="電子処方箋技術解説書">
  院内処方を行う医療機関のシステム上、院内処方を行う医薬品の名称や用法・用量等が記録され、一定のファイル形式で電子的に作成・保存されたもの。院内処方における処方の情報だけでなく、調剤や投薬の情報も含む。
</GlossaryEntry>

<GlossaryEntry term="院内処方等情報提供ファイル" reading="いんないしょほうとうじょうほうていきょうふぁいる" englishTerm="In-Hospital Prescription Information File" source="電子処方箋技術解説書">
  医療機関が院内処方等情報を記録し、電子処方箋管理サービスに登録するファイルを指す。
</GlossaryEntry>

<GlossaryEntry term="WAI-ARIA" reading="うぇいありあ" englishTerm="Web Accessibility Initiative - Accessible Rich Internet Applications" source="ウェブアクセシビリティ導入ガイドブック">
  W3Cで策定している技術仕様。HTMLだけではコンテンツの変化やUIの状態変化を伝えられない場合に、HTMLに加えて使用することで、ページ移動を伴わない動的なページ内コンテンツの変化を、スクリーンリーダーが認識して利用者に伝達できるように補完する。
</GlossaryEntry>

<GlossaryEntry term="WAIC" reading="うぇいく" englishTerm="Web Accessibility Infrastructure Committee" source="ウェブアクセシビリティ導入ガイドブック">
  情報通信アクセス協議会ウェブアクセシビリティ基盤委員会（Web Accessibility Infrastructure Committee）。WCAGの日本語訳やJIS X 8341-3:2016関連のガイドラインを公開している。
</GlossaryEntry>

<GlossaryEntry term="ウェブアクセシビリティ" reading="うぇぶあくせしびりてぃ" englishTerm="Web Accessibility" source="ウェブアクセシビリティ導入ガイドブック">
  利用者の障害の有無やその程度、年齢や利用環境にかかわらず、ウェブで提供されている情報やサービスを利用できること、またはその到達度。
</GlossaryEntry>

<GlossaryEntry term="HPKIカード" reading="えいちぴーけーあいかーど" englishTerm="HPKI Card" source="電子処方箋技術解説書">
  保健医療福祉分野の公開鍵基盤（Healthcare Public Key Infrastructure）に基づくICカード。医師・薬剤師の電子署名に使用される。ローカル署名ではHPKIカードのみが利用可能。
</GlossaryEntry>

<GlossaryEntry term="HPKIセカンド電子証明書リモート署名" reading="えいちぴーけーあいせかんどでんししょうめいしょりもーとしょめい" englishTerm="HPKI Second Electronic Certificate Remote Signature" source="電子処方箋技術解説書">
  カードレス署名の方法の1つであり、一般財団法人医療情報システム開発センター（MEDIS）が管理する鍵管理サービスのセカンド電子証明書を利用した電子署名。
</GlossaryEntry>

<GlossaryEntry term="枝番" reading="えだばん" englishTerm="Branch Number" source="オンライン資格確認等技術解説書">
  個人を識別可能とするため、被保険者資格に係る記号・番号（世帯単位）に付与する2桁の番号。
</GlossaryEntry>

<GlossaryEntry term="音声解説" reading="おんせいかいせつ" englishTerm="Audio Description" source="ウェブアクセシビリティ導入ガイドブック">
  登場人物の動作や表情、場面の移り変わりなど視覚的な情報を音声で伝えるもの。
</GlossaryEntry>

<GlossaryEntry term="オンライン資格確認等" reading="おんらいんしかくかくにんとう" englishTerm="Online Qualification Verification Services" source="オンライン資格確認等技術解説書">
  以下のサービスの総称。(1)オンライン資格確認サービス、(2)薬剤情報閲覧サービス、(3)特定健診情報閲覧サービス、(4)レセプト振替サービス。
</GlossaryEntry>

<GlossaryEntry term="オンライン資格確認等システム" reading="おんらいんしかくかくにんとうしすてむ" englishTerm="Online Qualification Verification System" source="電子処方箋技術解説書 / オンライン資格確認等技術解説書">
  オンライン資格確認システム、薬剤情報閲覧システム、特定健診情報閲覧システム、レセプト振替システムの総称。加入者（患者）が医療機関・薬局を受診する際、マイナンバーカードの利用者証明用電子証明書、または資格確認書を用いて、オンラインで資格確認を行うシステム。
</GlossaryEntry>

<GlossaryEntry term="オンライン資格確認等連携ソフト" reading="おんらいんしかくかくにんとうれんけいそふと" englishTerm="Online Qualification Verification Linkage Software" source="電子処方箋技術解説書 / オンライン資格確認等技術解説書">
  社会保険診療報酬支払基金が提供する、被保険者番号を利用した資格確認、照会番号の登録、薬剤情報・特定健診情報の取得等を行うためのアプリケーションソフト。電子処方箋管理サービスの利用にあたっても本ソフトを利用する。
</GlossaryEntry>

<GlossaryEntry term="オンライン資格確認用電子証明書" reading="おんらいんしかくかくにんようでんししょうめいしょ" englishTerm="Online Qualification Verification Electronic Certificate" source="オンライン資格確認等技術解説書">
  医療機関・薬局がオンライン資格確認等システム利用時の機関認証に用いる電子証明書。
</GlossaryEntry>

<GlossaryEntry term="オンライン資格確認用Webサービス（マイナ在宅受付Web）" reading="おんらいんしかくかくにんようウェブサービス（まいなざいたくうけつけウェブ）" englishTerm="Online Qualification Verification Web Service" source="電子処方箋技術解説書">
  オンライン診療・オンライン服薬指導において、マイナンバーカードを利用した資格確認・診療/薬剤情報・特定健診情報の閲覧同意等を行うためのWebサービス。
</GlossaryEntry>

<GlossaryEntry term="オンライン請求ネットワーク" reading="おんらいんせいきゅうねっとわーく" englishTerm="Online Billing Network" source="電子処方箋技術解説書 / オンライン資格確認等技術解説書">
  医療機関・薬局と審査支払機関を結ぶ閉域ネットワーク。レセプトのオンライン請求やオンライン資格確認等システム、電子処方箋管理サービスへの接続に利用される。IP-VPN接続方式またはIPsec+IKE接続方式で接続する。
</GlossaryEntry>

<GlossaryEntry term="オンライン請求用電子証明書" reading="おんらいんせいきゅうようでんししょうめいしょ" englishTerm="Online Billing Electronic Certificate" source="オンライン資格確認等技術解説書">
  医療機関・薬局がオンライン請求システム利用時の機関認証に用いる電子証明書。
</GlossaryEntry>

<GlossaryEntry term="介護情報基盤" reading="かいごじょうほうきばん" englishTerm="Long-term Care Information Platform" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  介護分野の情報を管理・共有するための基盤システム。電子カルテ標準仕様書では、介護情報基盤「外部インタフェース仕様書」に規定されたインターフェイスとの接続機能を有することが推奨される。
</GlossaryEntry>

<GlossaryEntry term="顔認証" reading="かおにんしょう" englishTerm="Facial Authentication" source="オンライン資格確認等技術解説書">
  顔によって個人を認識し、認証を行う仕組み。
</GlossaryEntry>

<GlossaryEntry term="顔認証付きカードリーダー" reading="かおにんしょうつきかーどりーだー" englishTerm="Face Authentication Card Reader" source="電子処方箋技術解説書 / オンライン資格確認等技術解説書">
  PIN無し認証を行うに当たり、マイナンバーカードの写真データを用いて顔認証を行うカードリーダー。患者の顔と実物のマイナンバーカードのICチップ内の顔画像を照合し認証する。患者が処方箋発行形態の選択や、薬局での調剤対象の処方箋選択などを行うことができる。
</GlossaryEntry>

<GlossaryEntry term="鍵管理サービス" reading="かぎかんりさーびす" englishTerm="Key Management Service" source="電子処方箋技術解説書">
  支払基金・国保中央会以外の主体が電子署名に必要となる電子証明書を管理するシステム。カードレス署名を行うにあたっては、鍵管理サービスに接続し、同サービスに格納される自身の電子証明書を用いて電子署名を行う。
</GlossaryEntry>

<GlossaryEntry term="確定前院内処方等情報" reading="かくていまえいんないしょほうとうじょうほう" englishTerm="Pre-finalized In-Hospital Prescription Information" source="電子処方箋技術解説書">
  院内処方を行う医療機関において、院内処方等情報が確定する前の情報を指す。重複投薬等チェックなどを行い、処方や調剤を行う薬剤を確定したものを院内処方等情報という。
</GlossaryEntry>

<GlossaryEntry term="確定前処方箋情報" reading="かくていまえしょほうせんじょうほう" englishTerm="Pre-finalized Prescription Information" source="電子処方箋技術解説書">
  重複投薬等チェックを行う前など、処方内容が確定する前の処方箋を指す。なお、重複投薬等チェック結果を踏まえて確定した処方箋の内容を「処方箋情報」という。
</GlossaryEntry>

<GlossaryEntry term="確定前調剤結果情報" reading="かくていまえちょうざいけっかじょうほう" englishTerm="Pre-finalized Dispensing Result Information" source="電子処方箋技術解説書">
  重複投薬等チェックを行う前など、確定する前の調剤結果情報を指す。なお、重複投薬等チェック結果を踏まえて確定した処方箋の内容を「処方箋情報」という。
</GlossaryEntry>

<GlossaryEntry term="紙の処方箋" reading="かみのしょほうせん" englishTerm="Paper Prescription" source="電子処方箋技術解説書">
  電子処方箋に対応する医療機関で患者が紙の処方箋を選択した場合、又は医療機関が電子処方箋に対応していない場合に発行される紙の処方箋。
</GlossaryEntry>

<GlossaryEntry term="カードレス署名" reading="かーどれすしょめい" englishTerm="Cardless Signature" source="電子処方箋技術解説書">
  支払基金・国保中央会以外の主体が提供する鍵管理サービスに格納される電子証明書（セカンド電子証明書）を利用して電子署名を付与する方法。鍵管理サービスとの接続にはWebAPI方式による通信が必要。
</GlossaryEntry>

<GlossaryEntry term="ガバメントクラウド" reading="がばめんとくらうど" englishTerm="Government Cloud" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  政府が提供するクラウドサービス基盤。電子カルテ標準仕様書において、ガバメントクラウド対象クラウドサービスを利用している場合、セキュリティ認証やアーキテクチャ要件の一部が充足されたものとみなされる。
</GlossaryEntry>

<GlossaryEntry term="キャプション" reading="きゃぷしょん" englishTerm="Caption" source="ウェブアクセシビリティ導入ガイドブック">
  映像コンテンツに付与される字幕。会話の内容や重要な効果音などが伝わるように提供する。
</GlossaryEntry>

<GlossaryEntry term="共通算定モジュール" reading="きょうつうさんていもじゅーる" englishTerm="Common Calculation Module" source="電子カルテ標準仕様書（診療所向け・病院向け）">
  診療報酬改定DXの一環として提供される、診療報酬の算定ロジックを共通化したモジュール。電子カルテとレセプトコンピュータの連携において、共通算定モジュール外部インタフェース利用ガイドに準拠した接続が推奨される。
</GlossaryEntry>

<GlossaryEntry term="クライアントアダプタサービス" reading="くらいあんとあだぷたさーびす" englishTerm="Client Adapter Service" source="電子処方箋技術解説書">
  一般財団法人医療情報システム開発センター（MEDIS）が管理する鍵管理サービスのセカンド電子証明書を利用したカードレス署名を行うにあたり、認証及びアクセストークンの取得・管理等の機能を担うライブラリ。電子カルテシステムや薬局システムに組み込むことができる。
