# システム機能一覧

主治医意見書電送サービスにおいて、医療機関のシステムベンダが実装する機能と、オンライン資格確認等連携ソフトが提供する機能の一覧です。

## 主治医意見書等作成ソフトの機能

### 意見書関連機能

<RequirementCard
  id="OL-F-001"
  title="主治医意見書情報登録機能"
  type="参考"
  section="意見書"
>
  主治医意見書情報を登録する機能。医師が主治医意見書等作成ソフト等の画面上で意見書情報を入力し保存します。

  画面上に「外部インタフェース仕様書」にある項目が存在しない場合、入力項目及び保存項目を追加改修します。既存項目においても「外部インタフェース仕様書」にある桁数、属性、省略不可等の条件に合わせて入力チェックを見直します。
</RequirementCard>

<RequirementCard
  id="OL-F-002"
  title="主治医意見書情報ファイル送信機能"
  type="参考"
  section="意見書"
>
  電子カルテシステムや文書作成システムから出力した主治医意見書情報ファイルを、本サービスへ連携する機能（オンライン資格確認等連携ソフトが参照するフォルダにファイルを配置する）。

  画面等で指定された主治医意見書より主治医意見書情報ファイル（XML形式、内部にCSVをBase64エンコードして格納）を作成し、連携アプリケーションから本サービスを通じて介護情報基盤に送信します。電子署名の付与は任意です。
</RequirementCard>

### 請求書関連機能

<RequirementCard
  id="OL-F-003"
  title="主治医意見書請求書情報登録機能"
  type="参考"
  section="請求書"
>
  主治医意見書の請求書情報を登録する機能。医師等が主治医意見書等作成ソフト等の画面上で請求書情報を入力し保存します。

  画面上に「外部インタフェース仕様書」にある項目が存在しない場合、入力項目及び保存項目を追加改修します。既存項目においても「外部インタフェース仕様書」にある桁数、属性、省略不可等の条件に合わせて入力チェックを見直します。
</RequirementCard>

<RequirementCard
  id="OL-F-004"
  title="主治医意見書請求書情報ファイル送信機能"
  type="参考"
  section="請求書"
>
  電子カルテシステムや文書作成システムから出力した主治医意見書請求書情報ファイルを、本サービスへ連携する機能（オンライン資格確認等連携ソフトが参照するフォルダにファイルを配置する）。

  画面等で指定された主治医意見書請求書より主治医意見書請求書情報ファイル（CSV形式）を作成します。いずれかの方法でファイル送信を行えるようにします。システムベンダにおいては、医療機関がいずれかの電送方式を選択できるようにシステム実装します。
</RequirementCard>

## オンライン資格確認等連携ソフトの機能

以下の機能は、社会保険診療報酬支払基金が提供するオンライン資格確認等連携ソフトが担う機能です。

<RequirementCard
  id="OL-OQS-001"
  title="主治医意見書情報ファイル（または請求書情報ファイル）連携機能"
  type="参考"
  section="連携ソフト"
>
  医療機関の主治医意見書等作成ソフト等で作成した主治医意見書情報ファイル（または主治医意見書請求書情報ファイル）を本サービスに連携するための機能。資格確認端末の所定のフォルダに格納された連携対象ファイルを本サービスへ送信します。
</RequirementCard>

<RequirementCard
  id="OL-OQS-002"
  title="主治医意見書情報ファイル（または請求書情報ファイル）連携結果取得機能"
  type="参考"
  section="連携ソフト"
>
  医療機関の主治医意見書等作成ソフト等で作成した主治医意見書情報ファイル（または主治医意見書請求書情報ファイル）を本サービスに連携した結果を取得するための機能。本サービスから連携結果を取得し、資格確認端末の所定のフォルダに保存します。
</RequirementCard>

## 既存システムの改修項目

医療機関において本サービスを利用するために、既存システムに対して以下の改修が必要です。

<RequirementCard
  id="OL-M-001"
  title="主治医意見書情報登録機能の追加変更（必須）"
  type="参考"
  section="改修"
>
  主治医意見書情報を登録する機能の追加変更。「外部インタフェース仕様書」にある項目が既存の主治医意見書情報登録画面にない項目について、必要に応じて画面項目、データベース項目を追加し、管理すること。

  - 「外部インタフェース仕様書」にある項目で既存画面にないものを追加
  - 画面項目、データベース項目の追加管理
  - 既存入力項目の桁数・属性・省略不可等の条件に合わせた入力チェック変更
</RequirementCard>

<RequirementCard
  id="OL-M-002"
  title="主治医意見書情報ファイル送信機能の追加（必須）"
  type="参考"
  section="改修"
>
  主治医意見書情報ファイルを作成し、本サービスへ送信する機能の追加。

  - CSV形式の主治医意見書情報の作成機能実装
  - 電子署名の付与機能の実装は任意
  - ローカル署名実装時は記録条件仕様書の規格に従い電子署名機能を実装するか、モジュールを購入
  - 資格確認端末経由：資格確認端末の所定のフォルダに作成ファイルを格納する機能を実装
</RequirementCard>
